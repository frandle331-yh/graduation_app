## サービス概要

複数のAIキャラクターが、指定されたお題やシチュエーションに沿って会話を行う様子を閲覧できるサービスです。
ユーザーはAIと対話するのではなく、AI同士の自然な会話を「盗み聞き」する体験を楽しめます。
新しいエンタメに興味のあるユーザー向けの「合法的に他人の会話を盗み聞きできる」を実現します。

## このサービスへの思い・作りたい理由

このサービスを私自身が作りたいと考えたきっかけは、明確な答えを出すことが難しい話題（例　政治、時事問題、思想的な問題）等に関して、自分がディベートに参加せずに客観的に議論展開を観察したいという発想が根本にあります。

そういった議論を活性化させ、自分の中に今までになかった発想や、視点を楽しみたいという私の個人的好奇心のもとに作りたいと思っています。



## ユーザー層について

・新しいもの好き・AIエンタメに興味のあるユーザー
・雑談配信のネタを探しているユーザー
・議論を楽しみたいユーザー

これらの層はアプリケーションの特性と強くマッチすると考えます。

## サービスの利用イメージ

ユーザーは会話のお題やシチュエーション、AIキャラクターの設定を選択します。
例：

キャラA：
役割：彼女
性格：感情的、嫉妬しやすい
口調：強め、タメ口

キャラB：
役割：彼氏
性格：理屈っぽい、やさしい
口調：落ち着いた、タメ口

会話の内容：デートで行きたいところについて

その後、AI同士が自動で会話を進行し、ユーザーはそれをリアルタイムで閲覧します。

自分が会話に参加する必要がないため、
作業中のBGM代わりや、配信のリアクション素材として気軽に利用できます。
「会話を覗き見る」という体験を通じて、
従来の対話型AIとは異なるエンタメ価値を得られます。

## ユーザーの獲得について

X（旧Twitter）での会話ログの一部切り抜き投稿
配信者による配信利用等
AIキャラクターやシチュエーションの面白さを前面に出した短尺動画展開

## サービスの差別化ポイント・推しポイント

### 差別化ポイント
ユーザーとAIが会話しない
AI同士の会話を「第三者視点」で楽しむ

既存の対話型AIサービスは、ユーザーの入力が体験の中心ですが、
本サービスでは「観察すること」自体が体験価値となります。

### 推しポイント

盗み聞きという人間の本能的な興味を活かしたコンセプト
シチュエーション次第で無限にコンテンツが生成される拡張性
配信・エンタメ用途に強く、二次利用しやすい構造

## 機能候補

### MVPリリースまでに実装予定

会話シチュエーション設定機能
AIキャラクター（2体）の簡易設定
AI同士のターン制会話生成
会話ログのリアルタイム表示
会話の再生成機能

### 本リリースまでに実装予定

認証機能の実装
キャラクター保存・管理機能
複数キャラクター（3体以上）対応
会話履歴の保存・共有（ユーザー紐づけ）
人気シチュエーションのランキング表示
